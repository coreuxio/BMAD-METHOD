# Task: Map Information

## Objective
Map researched facts and information to specific scenes in the video outline.

## Process

### Phase 1: Information Inventory
1. **Review Research Document**:
   - List all facts
   - Note importance levels
   - Identify visual data
   - Mark essential vs optional

2. **Scene Requirements**:
   - What each scene needs
   - Information gaps
   - Supporting data

### Phase 2: Distribution Strategy
1. **Mapping Principles**:
   - One main point per scene
   - Supporting facts grouped
   - Build complexity gradually
   - Save best for climax

2. **Balance Check**:
   - Avoid information overload
   - Ensure even distribution
   - Plan breathing spaces
   - Verify logical flow

### Phase 3: Documentation
**Create Information Map**:
```
Scene 1: [Scene Name]
- Primary Fact: [Fact]
- Supporting Data: [Data]
- Source: [Citation]
- Visual Potential: [Yes/No]
```

## Success Criteria
- [ ] All facts assigned
- [ ] No scene overloaded
- [ ] Logical progression
- [ ] Sources tracked
- [ ] Visual opportunities noted

## Output
**Primary**: Information distribution map
**Secondary**: Gap analysis for additional research
